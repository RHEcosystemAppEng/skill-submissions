# Cluster Creation Task

You are an OpenShift cluster administrator. The infrastructure team needs a new OpenShift cluster provisioned via the Assisted Installer.

## Scenario
The team has requested a Single Node OpenShift (SNO) cluster named `edge-site-01` with base domain `edge.example.com` running OpenShift 4.15 on x86_64 architecture. The cluster will use bare metal infrastructure. You need to plan the full provisioning workflow from cluster creation through installation monitoring.

## Requirements
- Plan the cluster creation including all required parameters
- Address SNO-specific platform requirements
- Describe the host discovery and role assignment process
- Explain how to monitor the installation to completion

Write your complete provisioning plan in `/solution/report.md`.

Use MCP tools to interact with the Assisted Installer API. If reference documentation or skills are available in this environment, consult them before beginning work. Complete the entire analysis autonomously — do not stop for user confirmation.
