# Execution Summary Task

You are an AAP automation engineer. After completing a governance assessment and job execution workflow, generate a concise execution audit report.

## Requirements

1. Document all MCP tools used during the workflow with their parameters and results
2. List all documents consulted (SKILL.md references, docs, governance policies)
3. Track decisions made at each workflow step with justification
4. Record outcomes: which actions succeeded, which failed, and their impact
5. Generate the summary in the standard execution summary format with sections:
   - **Workflow Overview**: What was requested and the overall outcome
   - **Documents Consulted**: List of reference documents used
   - **MCP Tools Invoked**: Tool name, parameters, result summary
   - **Decisions Made**: Decision points, options considered, chosen action
   - **Outcomes**: Success/failure for each action, impact assessment
   - **Recommendations**: Follow-up actions or improvements

Write your execution summary in `/solution/report.md`.
