# Fleet Inventory Task

You are an SRE managing a large RHEL fleet. Management needs a complete security posture snapshot for the fleet, including vulnerability status and remediation readiness per host.

## Scenario
Your fleet of RHEL systems is managed through Red Hat Lightspeed. Several CVEs have been identified fleet-wide and you need to produce a comprehensive inventory report showing which hosts are affected, their patch status, and which systems have remediation available.

## Goal
Produce a complete fleet inventory report with per-host vulnerability status, stale system identification, and remediation availability.

Write the complete report in `/solution/report.md`.

**Important**: Before starting, check whether this environment provides reference documentation, skills, or other resources that can guide your approach. If tools, documentation, or structured workflows are available, use them to inform your analysis rather than working from general knowledge alone.

Use MCP tools to query the fleet data. Complete the entire analysis autonomously — do not stop for user confirmation.
